# SUMOVI---Sensor-Ultrass-nico-para-Mobilidade-de-Deficientes-Visuais-com-IoT

Sensor Ultrassônico para Mobilidade de Deficientes Visuais com IoT.

## Objetivo

Desenvolver um dispositivo assistivo de baixo custo capaz de detectar obstáculos e auxiliar a mobilidade de pessoas com deficiência visual.

## Tecnologias

- ESP32
- Sensores ultrassônicos HC-SR04
- Sensor DS18B20
- KiCad
- Impressão 3D

## Funcionalidades

- Detecção de obstáculos
- Compensação térmica
- Feedback sonoro
- Arquitetura preparada para IoT

## Status

Em desenvolvimento.
